# 🎓 India's Graduate Skill Index - Career Accelerator

## 📌 Project Overview

**Graduate Skill Index - Career Accelerator** is a responsive landing page designed to address the gap between education and employability among graduates in India.

The platform focuses on helping students develop industry-ready skills, build real-world projects, prepare for interviews, and receive career guidance.

---

## 🚀 Features

### 💡 Skill Development
- Learn industry-relevant skills
- Focus on practical knowledge required by companies
- Improve career readiness

### 🛠 Real Projects
- Gain hands-on experience
- Build a strong professional portfolio
- Apply skills to real-world scenarios

### 🎯 Interview Preparation
- Mock interview practice
- Improve communication skills
- Increase confidence for job opportunities

### 👨‍🏫 Career Guidance
- Personalized career paths
- Expert mentorship
- Better decision-making for career growth
